## Factor-Based Equity Strategy

This project implements a multi-factor equity trading strategy using momentum and volatility signals, with monthly rebalancing and benchmark comparison against the S&P 500 (SPY).

### Features
- Factor construction and ranking
- Portfolio backtesting
- Risk-adjusted performance metrics
- Equity curve, drawdown, and rolling Sharpe plots

### Disclaimer
This project is for educational purposes only and does not constitute investment advice.
